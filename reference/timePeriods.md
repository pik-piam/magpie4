# timePeriods

Calculates MAgPIE time period lengths between each two timesteps

## Usage

``` r
timePeriods(gdx)
```

## Arguments

- gdx:

  GDX file

## Value

magpie time periods as MAgPIE object as a number of years

## Author

Mishko Stevanovic

## Examples

``` r
  if (FALSE) { # \dontrun{
    x <- timePeriods(gdx=gdx)
  } # }
```
