# ValueMaterialDemand

calculates agricultural costs without taxes and incentives (i.e. GHG
taxes and BII incentives)

## Usage

``` r
ValueMaterialDemand(gdx, file = NULL, level = "regglo")
```

## Arguments

- gdx:

  GDX file

- file:

  a file name the output should be written to using write.magpie

- level:

  aggregation level, reg, glo or regglo
