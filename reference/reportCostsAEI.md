# reportCostsAEI

reports MAgPIE AEI costs

## Usage

``` r
reportCostsAEI(gdx)
```

## Arguments

- gdx:

  GDX file

## Value

magpie object containing AEI costs

## AEI cost variables

|            |                     |                                        |
|------------|---------------------|----------------------------------------|
| Name       | Unit                | Meta                                   |
| Costs\|AEI | million US\$2017/yr | Costs for area equipped for irrigation |
