# FoodDemandModuleConsumerPrices

Calculates food prices that enter demand module

## Usage

``` r
FoodDemandModuleConsumerPrices(gdx, level = "iso", valueAdded = FALSE)
```

## Arguments

- gdx:

  GDX file

- level:

  reg or iso

- valueAdded:

  whether to add the value-added marketing margin to the total
  expenditures

## Value

magpie object
