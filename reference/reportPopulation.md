# reportPopulation

reports Population

## Usage

``` r
reportPopulation(gdx, level = "regglo")
```

## Arguments

- gdx:

  GDX file

- level:

  spatial aggregation: "reg", "glo", "regglo", "iso"

## Value

population as MAgPIE object

## Population variables

|            |                |                  |
|------------|----------------|------------------|
| Name       | Unit           | Meta             |
| Population | million people | Total population |
