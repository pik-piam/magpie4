# NitrogenBudgetNonagland

calculates projections of Nitrogen Budgets for non-agricutlural land
from a MAgPIE gdx file

## Usage

``` r
NitrogenBudgetNonagland(gdx, level = "reg")
```

## Arguments

- gdx:

  GDX file

- level:

  aggregation level, reg, glo or regglo
