# outputCheck

Function to check a MAgPIE gdx file for known problems (e.g. non-zero
dummy variables). The function will throw warnings for problem found in
the outputs.

## Usage

``` r
outputCheck(gdx)
```

## Arguments

- gdx:

  GDX file

if (FALSE) { # \dontrun{
outputCheck(gdx)
