# changelogVariables

Returns a named vector of variables to put into the data changelog, see
[`addToDataChangelog`](addToDataChangelog.md).

## Usage

``` r
changelogVariables()
```

## See also

Other Data Changelog: [`addToDataChangelog()`](addToDataChangelog.md),
[`changelogYears()`](changelogYears.md)
