# modelstat

MAgPIE model stat of all optimizations - main optimization and (if used)
presolve optimization.

## Usage

``` r
modelstat(gdx, file = NULL)
```

## Arguments

- gdx:

  GDX file

- file:

  a file name the output should be written to using write.magpie

## Value

A MAgPIE object containing the modelstat

``` r
  if (FALSE) { # \dontrun{
    x <- modelstat(gdx)
  } # }
```
