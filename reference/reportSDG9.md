# reportSDG9

reports all SDG indicators relevant for SD9 - Industrial innovation and
infrastructure

## Usage

``` r
reportSDG9(gdx, level = "regglo")
```

## Arguments

- gdx:

  GDX file

- level:

  An aggregation level for the spatial dimension. Can be any level
  available via superAggregateX.

## Value

MAgPIE object
