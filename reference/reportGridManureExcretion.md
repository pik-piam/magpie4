# reportGridManureExcretion

reports Manure with reprting names on grid level.

## Usage

``` r
reportGridManureExcretion(gdx)
```

## Arguments

- gdx:

  GDX file

## Value

MAgPIE object

## Grid-level manure excretion

This function produces grid-level (0.5 degree) manure excretion and
management data. Includes total manure, breakdown by AWMS and livestock
type, and losses from confinement.
