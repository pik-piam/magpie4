# wageRent

calculates wage rent for exogenous wage scenarios

## Usage

``` r
wageRent(gdx, file = NULL, level = "regglo")
```

## Arguments

- gdx:

  GDX file

- file:

  a file name the output should be written to using write.magpie

- level:

  aggregation level, reg, glo or regglo
