# malmquist

calcluates malmquist index based on a MAgPIE gdx file

## Usage

``` r
malmquist(gdx, file = NULL)
```

## Arguments

- gdx:

  GDX file

- file:

  a file name the output should be written to using write.magpie

## Value

A MAgPIE object containing the malmquist index
