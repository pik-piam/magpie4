# reportBiochar

reports production and carbon storage of biochar

## Usage

``` r
reportBiochar(gdx, level = "regglo")
```

## Arguments

- gdx:

  GDX file

- level:

  The aggregation level to be used ("regglo" by default)

## Value

Biochar production and carbon storage as MAgPIE object
