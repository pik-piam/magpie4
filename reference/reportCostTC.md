# reportCostTC

reports MAgPIE TC costs

## Usage

``` r
reportCostTC(gdx)
```

## Arguments

- gdx:

  GDX file

## Value

magpie object with TC costs

## TC cost variables

|           |                     |                                       |
|-----------|---------------------|---------------------------------------|
| Name      | Unit                | Meta                                  |
| Costs\|TC | million US\$2017/yr | Technological change investment costs |
